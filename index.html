What is JetApps?

JetApps is an platform for developers and sysadmins, you can to build and run applications. JetApps provide containers and is compatible with Docker's containers.


What is Docker?

Docker is an open-source project to easily create lightweight, portable, self-sufficient containers from any application. The same container that a developer builds and tests on a laptop can run at scale, in production, on VMs, bare metal, OpenStack clusters, public clouds and more.


Support

etApps comes with support for Python, Ruby, Node.js, Go and more! Services like PostgreSQL, mongoDB, MySQL and pipeline steps. Also, JetApps has support for Docker containers.
